## 技术栈：uni-app

## How to start?
+ `npm install`
+ `npm run dev:%PLATFORM%`
+ `npm run build:%PLATFORM%`

`%PLATFORM%` 可取值如下：
|值	|平台|
|:----|:----:|
|h5|	H5|
|mp-alipay|	支付宝小程序|
|mp-baidu|	百度小程序|
|mp-weixin	|微信小程序|
|mp-toutiao|	字节跳动小程序|
|mp-qq | qq小程序|
